How-to-use

To generate text with Spectrum first thing you must do is pick a seed text, seed text might be anything a word, sentence, or a paragraph something you believe will make sense to AI after that you can press the generate button and see AI's prediction about your seed.

About Spectrum

Spectrum is an AI that uses deep learning to generate rap song lyrics you can see the live demo from your browser type a seed text and generate rap songs around 1000 characters. If you want to know how to use the web app you can check out the how-to-use section above.

Spectrum is built with

  • Python
  • TensorFlow & Keras
  • Flask
  • Bootstrap

API

Spectrum has a free web API to learn about example API request, sample response and more you can check out the Github repo
